# ExoMol Database

The 2024 release describes ExoMol as a molecular data resource for hot-atmosphere spectroscopy, reporting recommended line lists for 91 molecules and 224 isotopologues with almost 10^12 transitions. Source: [24TeYuZh](../../Raw/Sources/24TeYuZh.xml).
