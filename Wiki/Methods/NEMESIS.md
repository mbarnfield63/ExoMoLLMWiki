# NEMESIS

NEMESIS is an atmospheric retrieval and radiative transfer code used for Solar System and exoplanet atmosphere modelling. The Dozen CO2 line list paper generated ExoMolOP opacity tables compatible with NEMESIS. Source: [26YuBaBo](../../Raw/Sources/26YuBaBo.json).
