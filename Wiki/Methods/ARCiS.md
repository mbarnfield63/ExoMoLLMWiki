# ARCiS

ARCiS is an atmospheric retrieval and radiative transfer code used for exoplanet atmosphere modelling. The Dozen CO2 line list paper generated ExoMolOP opacity tables compatible with ARCiS. Source: [26YuBaBo](../../Raw/Sources/26YuBaBo.json).
