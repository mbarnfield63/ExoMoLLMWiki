# LLM Wiki Query

Use this skill when answering questions from the ExoMol LLM Wiki.

## Steps

1. Search `Wiki/catalog.jsonl` first.
2. Open matching notes under `Wiki/`.
3. Follow raw source links only when claim-level provenance is required.
4. Prefer concise answers with source-backed statements.
5. State when the wiki does not contain the requested value.

Do not infer missing line-list values, partition functions, transition frequencies, or citations.
